
# 📊 Portfolio Dashboard – Credit Risk & Financial Rating Analysis

<br>

This project is an interactive data analytics dashboard built with Plotly Dash, designed to analyze and visualize financial risk indicators, credit ratings, and mood-based classification of companies across multiple years.

The dashboard allows users to dynamically explore datasets loaded from Excel files and compare trends across different time periods and rating agencies.

#

### 📁 Data Sources

The application uses multiple Excel datasets:

- exemplary2019.xlsx
- exemplary2020.xlsx
- exemplary2021.xlsx
- exemplary2022.xlsx

Each file contains structured financial data including:

- Credit ratings (e.g. Moody’s, internal ratings)
- Risk classification (flagged / not flagged) is color-coded for visual clarity

#

### 📊 Dashboard Features

The dashboard provides interactive exploration of financial risk data through:

<br>

1. 📌 Rating Distribution Analysis

- Shows how credit ratings are distributed across companies
- Compares internal and Moody’s ratings
- Helps identify concentration of high/low credit quality entities

<br>

2. ⚠️ Risk Classification Overview

Visualizes risk flags assigned to entities:
- flagged vs non-flagged entities
- comparison across years
- highlights changes in risk exposure over time

<br>

3. 📈 Year-over-Year Comparison

Allows users to track how financial indicators evolve:

- trends in ratings stability
- changes in risk classification
- shifts in portfolio composition over time

<br>

4. 📊 Interactive Filtering

User interactions with dropdowns and radio buttons trigger updates to tables, charts, and aggregated views.

#

### 🧠 Technology Stack

- Python 3
- Plotly Dash
- Pandas
- NumPy
- Dash Bootstrap Components
- Dash Mantine Components (UI styling)
- Plotly Express & Graph Objects

#

### 🎯 Key Insights from the Dashboard

This tool enables:

- identification of high-risk financial entities
- comparison of credit ratings across agencies
- detection of inconsistencies in rating systems
- exploration of temporal trends in risk classification
- interactive financial data exploration without coding
